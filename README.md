# PureLogics Bootcamp - Lab Work Repository

Welcome to my official lab work repository for the **PureLogics Python & AI Bootcamp**. This repository contains all the practical exercises, assignments, and project files I will complete throughout the duration of the bootcamp.

## 📌 About the Bootcamp

The PureLogics Bootcamp is an intensive, 3-month, in-person training program focused on developing skills in Python programming, Artificial Intelligence, Machine Learning, and related technologies. 

- 📍 Location: Lahore, Pakistan (On-site)
- ⏳ Duration: 3 Months
- 💼 Organized by: PureLogics Pvt. Ltd.
- 💻 Areas of Focus:
  - Python Programming
  - Data Analysis and Preprocessing
  - Machine Learning (ML)
  - Deep Learning (DL)
  - Generative AI and Large Language Models (LLMs)
  - Real-world project development

## 🗂️ Repository Structure

This repository is organized on a **weekly basis**. Each week includes specific lab tasks and assignments provided by the instructors.


Each folder contains Python scripts, notebooks, and any relevant documentation or notes I have created during the labs.

## 📘 Objectives of This Repository

- To document my learning journey throughout the bootcamp
- To keep all lab work organized and easily accessible
- To track weekly progress and key learning outcomes
- To build a strong technical foundation for future AI and data science roles
